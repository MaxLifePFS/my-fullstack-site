One-Time Setup Needed

The room calendar stores bookings in your Supabase database, but its table hasn't been created yet. This takes about a minute:

  1. Open your Supabase SQL Editor
  2. Paste the SQL below and click Run
  3. Come back here and click Try Again
create table public.room_bookings (
  id uuid primary key default gen_random_uuid(),
  guest text not null,
  room text not null,
  check_in date not null,
  check_out date not null,
  created_at timestamptz default now()
);

alter table public.room_bookings enable row level security;

create policy "Owner full access" on public.room_bookings
  for all to authenticated using (true) with check (true);
